Theme Colors Customization

There are two ways to customize theme main colors. One way is to change primary and secondary theme color variables in the colortheme-[yourcolorshceme].scss file. This method will require you to re-compile the state template, you can find instructions in the readme file. After the compilation you will have updated colortheme-[yourcolorshceme].css file.

Another way is to manually find and replace all main theme colors and their variations in colortheme-[yourcolorshceme].css. Main theme colors classes will be listed at the top of the CSS file. You can copy their values and use find/replace feature in your web editor software to replace all those colors references with your custom colors throughout entire css file.

Note: Before changing your colors make sure that your custom theme colors have sufficient color cotrast ratios.
