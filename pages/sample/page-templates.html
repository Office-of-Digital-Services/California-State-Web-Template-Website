Page Templates

There are three page template options in the State Web Tempalte:

  • lsnding.njk (full width page)
  • page.njk (standard page)
  • navicons-landing.njk (full width that has main navigation with icons)
  • navicons.njk (has main navigation with icons)
  • search.njk (search results page)

To use use one of those option you need to specify layout key in your front matter data at the beginnig of your page. 

---
title: Your page title
layout: page.njk
---

Note: You can inspect page templates source code in /src/_includes/layouts/ folder or download source code zip file at the State Web Template GitHub releases page.

Using includes

You can build your page by using the State Web Template includes. Several common components (such as page headers, footers, navigation, etc.) are already included in pages using njk includes. You can also add html or njk includes to your apge like this:

❴% include "../../src/_includes/modules/header-primary-banner.html" %❵

Note: Path to the module include will be relative to the directory of your page.
