Forms elements

A form allows visitors to submit information to a server.

Common form elements include:

  • Labels
  • Text fields
  • Text areas
  • Checkboxes
  • Radio buttons
  • Dropdown menus (select)
  • File inputs

Label

The label should:

  • Describe the purpose of the text field.
  • Use sentence-style capitalizationâ€”only capitalize the first word.
  • Be short and concise.
  • Use nouns or short noun phrases.
  • Inform visitors whether or not the form input is required (represented by an asterisk).
  • Avoid instructional text like "Click to get started".
