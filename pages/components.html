Components

Components are the reusable building blocks of the California State Template.

Each component is designed to enhance user experiences, and has been created to work together to create web page patterns.

Our components were:

  • Created with performance and accessibility in mind
  • Designed to be more adaptable
  • Built with user needs in mind

All components

Use these components to present information and images.

Accordion

A list of headings that hide or reveal additional content when selected.

Alert

Keeps visitors informed of important and sometimes time-sensitive changes.

Blockquote

Emphasizes important speaker quotes or content excerpts.

Breadcrumb

Indicates the current page's location within a navigational hierarchy.

Button

Draws attention to your primary call to action with a large clickable surface.

Countdown timer

Emphasizes an important event by counting toward or away from a fixed point in time.

Form elements

Allows visitors to enter and submit information to a server.

Horizontal separator

A style component that uses a line to separate content into sections.

Lists

A vertical grouping of related content.

Modal

Displays content in a temporary window that appears over the page content.

Number counter

Adds rolling animation to numbers.

Parallax

Allows background and foreground elements to move at different speeds.

Progress bar

Displays the current status of a multi-step task.
